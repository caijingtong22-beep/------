# 从 `predict()` 开始理解整套调用顺序

你现在运行的是：

```python
tree = build_tree(dataset, max_depth=3, min_size=1)

for row in dataset:
    y_true = row[-1]
    y_pred = predict(tree, row)
    print(f"True: {y_true}, Predicted: {y_pred}")
```

这里其实分成两个阶段：

```text
阶段一：先构建树 build_tree()
阶段二：再用树预测 predict()
```

---

# 一、整体调用顺序

```text
build_tree(dataset)
    ↓
get_best_split(dataset)
    ↓
split(root)
    ↓
递归构建 left / right 子树
    ↓
得到完整 tree

然后：

predict(tree, row)
    ↓
根据当前 node 判断走 left 还是 right
    ↓
predict(node['left'], row)
    ↓
predict(node['right'], row)
    ↓
直到 node 不是 dict
    ↓
return 0 或 1
```

---

# 二、先构建树：`build_tree()`

调用：

```python
tree = build_tree(dataset, max_depth=3, min_size=1)
```

进入：

```python
def build_tree(train, max_depth, min_size):
    root = get_best_split(train)
    split(root, 0, max_depth, min_size)
    return root
```

执行顺序：

```text
1. get_best_split(train)
   找到根节点最佳划分

2. split(root, ...)
   从根节点开始递归长树

3. return root
   返回完整树
```

---

# 三、`get_best_split()` 的调用链

```python
root = get_best_split(train)
```

进入：

```python
def get_best_split(dataset):
```

它会尝试所有划分：

```text
color == 0
color == 1
color == 2

root == 0
root == 1
root == 2

knock == 0
knock == 1
knock == 2
```

每试一种，就调用：

```python
groups = split_dataset(index, val, dataset)
gini = gini_index(groups)
```

所以调用链是：

```text
get_best_split(dataset)
    ↓
split_dataset(index, value, dataset)
    ↓
得到 groups = (left, right)
    ↓
gini_index(groups)
    ↓
计算这次划分的基尼指数
    ↓
如果 gini 更小，就更新最佳划分
```

最后返回：

```python
{
    'index': best_index,
    'value': best_value,
    'groups': best_groups
}
```

例如：

```python
{
    'index': 0,
    'value': 0,
    'groups': (left, right)
}
```

表示：

```text
当前最优划分规则是：
color == 0
```

---

# 四、`split()` 开始递归长树

`build_tree()` 里执行：

```python
split(root, 0, max_depth, min_size)
```

此时：

```python
node = root
```

也就是：

```python
node =
{
    'index': 0,
    'value': 0,
    'groups': (left, right)
}
```

进入：

```python
def split(node, depth=0, max_depth=3, min_size=1):
```

---

## 第一步：取出左右数据集

```python
left, right = node['groups']
del node['groups']
```

原来：

```python
{
    'index': 0,
    'value': 0,
    'groups': (left, right)
}
```

变成：

```python
{
    'index': 0,
    'value': 0
}
```

---

## 第二步：判断是否停止

### 情况一：有一边为空

```python
if not left or not right:
    node['left'] = node['right'] = to_terminal(left + right)
    return
```

调用：

```python
to_terminal(left + right)
```

返回多数类，比如：

```python
1
```

---

### 情况二：达到最大深度

```python
if depth >= max_depth:
    node['left'], node['right'] = to_terminal(left), to_terminal(right)
    return
```

左右都变成叶节点。

---

### 情况三：左边样本太少

```python
if len(left) <= min_size:
    node['left'] = to_terminal(left)
```

否则继续划分：

```python
node['left'] = get_best_split(left)
split(node['left'], depth + 1, max_depth, min_size)
```

调用链：

```text
split(node)
    ↓
get_best_split(left)
    ↓
split_dataset(...)
    ↓
gini_index(...)
    ↓
返回左子节点
    ↓
split(node['left'])
```

---

### 情况四：右边样本太少

```python
if len(right) <= min_size:
    node['right'] = to_terminal(right)
```

否则继续划分：

```python
node['right'] = get_best_split(right)
split(node['right'], depth + 1, max_depth, min_size)
```

调用链：

```text
split(node)
    ↓
get_best_split(right)
    ↓
split_dataset(...)
    ↓
gini_index(...)
    ↓
返回右子节点
    ↓
split(node['right'])
```

---

# 五、构建树阶段的完整递归结构

可以理解成：

```text
build_tree(dataset)
│
├── get_best_split(dataset)
│       ├── split_dataset(...)
│       ├── gini_index(...)
│       ├── split_dataset(...)
│       ├── gini_index(...)
│       └── 返回 root
│
└── split(root)
        │
        ├── 处理 left
        │       ├── get_best_split(left)
        │       │       ├── split_dataset(...)
        │       │       └── gini_index(...)
        │       │
        │       └── split(node['left'])
        │               ├── get_best_split(...)
        │               └── split(...)
        │
        └── 处理 right
                ├── get_best_split(right)
                │       ├── split_dataset(...)
                │       └── gini_index(...)
                │
                └── split(node['right'])
                        ├── get_best_split(...)
                        └── split(...)
```

直到所有分支都变成：

```python
0
```

或者：

```python
1
```

也就是叶节点。

# 六、最终得到的 tree

最后可能得到类似：

```python
tree =
{
    'index': 0,
    'value': 0,

    'left': 1,

    'right': {
        'index': 1,
        'value': 2,

        'left': 0,
        'right': 1
    }
}
```

这棵树可以读成：

```python
if color == 0:
    predict 1
else:
    if root == 2:
        predict 0
    else:
        predict 1
```

# 七、开始预测：`predict(tree, row)`

调用：

```python
y_pred = predict(tree, row)
```

进入：

```python
def predict(node, row):
```

此时：

```text
node = tree
row = 当前样本
```

例如：

```python
row = [0, 0, 1, 1]
```

---

# 八、predict 的递归调用顺序

源码：

```python
def predict(node, row):
    if isinstance(node, dict):
        if row[node['index']] == node['value']:
            return predict(node['left'], row)
        else:
            return predict(node['right'], row)
    else:
        return node
```

---

# 九、第一层预测

假设：

```python
node =
{
    'index': 0,
    'value': 0,
    'left': 1,
    'right': {...}
}
```

判断：

```python
isinstance(node, dict)
```

结果是：

```python
True
```

说明当前还不是叶节点。

然后执行：

```python
row[node['index']] == node['value']
```

也就是：

```python
row[0] == 0
```

如果成立：

```python
return predict(node['left'], row)
```

也就是：

```python
predict(1, row)
```

---

# 十、第二层预测：进入叶节点

现在：

```python
node = 1
```

执行：

```python
isinstance(node, dict)
```

结果是：

```python
False
```

所以：

```python
return node
```

也就是：

```python
return 1
```

预测完成。

---

# 十一、如果第一层不满足条件

假设：

```python
row = [2, 1, 0, 0]
```

第一层：

```python
row[0] == 0
```

也就是：

```python
2 == 0
```

不成立。

于是：

```python
return predict(node['right'], row)
```

进入右子树。

---

假设右子树是：

```python
node =
{
    'index': 1,
    'value': 2,
    'left': 0,
    'right': 1
}
```

继续判断：

```python
row[node['index']] == node['value']
```

也就是：

```python
row[1] == 2
```

如果：

```python
1 == 2
```

不成立，就执行：

```python
predict(node['right'], row)
```

也就是：

```python
predict(1, row)
```

最后返回：

```python
1
```

---

# 十二、predict 的递归结构

```text
predict(tree, row)
│
├── node 是 dict
│
├── 判断 row[node['index']] == node['value']
│
├── 如果 True
│       ↓
│   predict(node['left'], row)
│
└── 如果 False
        ↓
    predict(node['right'], row)
```

如果下一层还是 dict：

```text
继续判断
继续往下走
```

如果下一层是数字：

```text
return 0 或 1
```

---

# 十三、预测阶段完整调用链

```text
for row in dataset:
    ↓
    y_true = row[-1]
    ↓
    y_pred = predict(tree, row)
                ↓
                predict(当前节点, row)
                    ↓
                    判断是不是 dict
                    ↓
                    判断样本是否满足当前节点规则
                    ↓
                    进入 left 或 right
                    ↓
                    predict(下一个节点, row)
                        ↓
                        ...
                            ↓
                            到达叶节点
                            ↓
                            return 0 或 1
    ↓
    print(y_true, y_pred)
```

# 十四、训练和预测的函数调用对照

## 训练阶段

```text

    ↓
get_best_split
    ↓
split_dataset
    ↓
gini_index
    ↓

    ↓

```


## 预测阶段


# 十五、最核心的一句话

```text

```
