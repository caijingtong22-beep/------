# 决策树训练阶段完整函数链：输入 / 输出详解

训练阶段主链：

```text
build_tree
    ↓
get_best_split
    ↓
split_dataset
    ↓
gini_index
    ↓
split
    ↓
to_terminal
```

注意：

这里不是严格“线性调用”。

真实情况更像：

```text
build_tree
    ├── get_best_split
    │       ├── split_dataset
    │       └── gini_index
    │
    └── split
            ├── get_best_split
            │       ├── split_dataset
            │       └── gini_index
            │
            └── to_terminal
```

下面按：

```text
输入是什么
输出是什么
内部干了什么
```

逐个分析。

---

# 1. build_tree

源码：

```python
def build_tree(train, max_depth, min_size):
    root = get_best_split(train)
    split(root, 0, max_depth, min_size)
    return root
```

---

## 输入

```python
train
```

即：

```python
dataset =
[
 [特征1,特征2,特征3,label],
 ...
]
```

例如：

```python
[
 [0,0,0,1],
 [1,0,1,0],
 ...
]
```

---

还有：

```python
max_depth
```

树最大深度。

例如：

```python
3
```

---

以及：

```python
min_size
```

叶节点最少样本数。

例如：

```python
1
```

---

## 输出

输出：

```python
tree
```

即：

```python
{
    'index':0,
    'value':0,
    'left':...,
    'right':...
}
```

完整决策树。

---

## build_tree 干了什么

### 第一步

```python
root = get_best_split(train)
```

找到：

```text
根节点最佳划分
```

---

### 第二步

```python
split(root,...)
```

递归长整棵树。

---

### 第三步

```python
return root
```

返回完整树。

# 5. split

源码：

```python
def split(node, depth=0, max_depth=3, min_size=1):
```

---

## 输入

### node

例如：

```python
{
    'index':0,
    'value':0,
    'groups':(...)
}
```

当前树节点。

---

### depth

当前树深度。

例如：

```python
0
```

表示根节点。

---

### max_depth

最大深度限制。

---

### min_size

最小样本限制。

---

## 输出

严格来说：

```text
没有 return
```

---

因为：

```python
split()
```

是：

# 原地修改 node

---

即：

输入前：

```python
{
    'index':0,
    'value':0,
    'groups':(...)
}
```

---

执行后：

```python
{
    'index':0,
    'value':0,

    'left':...,
    'right':...
}
```

node 被“长成树”。

---

## split 干了什么

### 1. 取出 groups

```python
left, right = node['groups']
```

---

### 2. 删除 groups

```python
del node['groups']
```

---

### 3. 判断是否停止

例如：

```python
if depth >= max_depth
```

---

### 4. 否则递归继续长树

```python
node['left'] = get_best_split(left)
split(node['left'], ...)
```

# 7. 整个训练阶段的数据流

```text
dataset
   ↓

build_tree(dataset)
   ↓

get_best_split(dataset)
   ↓

split_dataset(index,value,dataset)
   ↓

(left,right)
   ↓

gini_index(groups)
   ↓

gini值
   ↓

返回最佳划分node
   ↓

split(node)
   ↓

继续对子节点：
    get_best_split
        ↓
    split_dataset
        ↓
    gini_index
        ↓
    split

直到：
    to_terminal
        ↓
    返回0/1叶节点

最终：
生成完整tree
```

---

# 8. 每个函数最本质的一句话

| 函数 | 本质作用 |
|---|---|
| build_tree | 整棵树入口 |
| get_best_split | 找最佳划分 |
| split_dataset | 按规则切数据 |
| gini_index | 评价切得好不好 |
| split | 递归长树 |
| to_terminal | 生成叶节点类别 |
